Docker

Utilizando o Docker com o Apache

Inicialmente instale-o

sudo apt install docker docker.io -y

Baixe a imagem do Ubuntu/Apache:

sudo docker pull ubuntu/apache2

No nosso caso, os arquivos que serão hospedados estão em Documentos/GitHub/GitTutorials. Para lançá-los no Docker, você deve informar a origem (na sua máquina) e o destino (no Docker), da seguinte forma -v origem:destino. O "-v" indica que esse volume será compartilhado com o Docker.

sudo docker run -d -v /home/$USER/Documentos/GitHub/GitTutorials:/var/www/html -p 8080:80 ubuntu/apache2

Legenda dos atributos:

  • "-d": será rodado em background, deixando o terminal livre;
  • "-v": volume que será compartilhado com o Docker;
  • "-p": porta em que a página será exibida (nessa caso 8080).
